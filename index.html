Project Overview

This interactive election campaign map provides real-time visualization of polling booth locations across multiple municipalities. The platform includes detailed information about political party dominance, key personnel contacts, and visit tracking for efficient campaign management.

Key Features: Party-based filtering, municipality visualization with 7 distinct colors, personnel contact management, visit tracking analytics, and downloadable thematic maps.
